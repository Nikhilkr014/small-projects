# 🪐 Multiverse of Innovation: The 17-Project Lab

Welcome to my central creative hub. This repository serves as a live gallery housing **17 interactive web applications, tools, and experimental builds** managed from a single unified ecosystem. 

Each piece within this lab represents an exploration of modern frontend architectures, interactive motion design, and algorithmic problem-solving.

🔗 **[Explore the Live Showcase Gallery Here](https://Nikhilkr014.github.io/small-projects/)**

## 🚀 Featured Tech Stack

This laboratory leverages a modern, highly responsive engineering stack to power intensive interactive experiences:

*   **Core Systems:** HTML5 / Advanced Semantic Architecture, CSS3, Modern ES6+ JavaScript
*   **Motion & 3D Interactive Design:** GSAP (GreenSock Animation Platform), Three.js, Custom 3D Viewports
*   **Styling Architecture:** Tailwind CSS (Utility-first system), custom fluid typography, dynamic glow maps
*   **Hardware Integration:** Arduino Serial Data Flow Interfaces & IoT Micro-architectures

## 🎨 Interactive Visual Features

> ⚡ **Hover Effects & Trackers:** The hub features hardware-accelerated pointer tracking that maps custom glowing vectors directly to user input, avoiding performance degradation even on budget viewports.

> 🛠️ **Vite Compilation:** The underlying subdirectory (`/vanguard/`) uses an advanced subfolder asset-hosting bundle mechanism via Vite configurations, optimizing performance across asset-heavy builds.

## 🛠️ Local Development & Replication

Want to run the complete environment locally? Clone the repository and initialize the build matrix:

```bash
# Clone the system repository
git clone [https://github.com/Nikhilkr014/small-projects.git](https://github.com/Nikhilkr014/small-projects.git)

# Navigate to the project base
cd small-projects

# Sync to the Vanguard development subdirectory
cd vanguard
npm install

# Compile the bundle configuration
npm run build
